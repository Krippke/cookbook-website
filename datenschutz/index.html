Datenschutzerklärung

1. Datenschutz auf einen Blick

Allgemeine Hinweise

Die folgenden Hinweise geben einen einfachen Überblick darüber, was mit Ihren personenbezogenen Daten passiert, wenn Sie diese Website besuchen.

Datenerfassung auf dieser Website

Wer ist verantwortlich für die Datenerfassung auf dieser Website?

Die Datenverarbeitung auf dieser Website erfolgt durch den Websitebetreiber. Dessen Kontaktdaten können Sie dem Impressum dieser Website entnehmen.

Wie erfassen wir Ihre Daten?

Diese Website erfasst KEINE personenbezogenen Daten. Wir verwenden:

  • Keine Cookies
  • Keine Tracking-Tools
  • Keine Analyse-Software
  • Keine Werbenetzwerke

Die Website wird als statische Seite bereitgestellt.

2. Hosting

Diese Website wird extern gehostet. Bei der Bereitstellung der Website können technisch bedingt folgende Daten verarbeitet werden:

  • IP-Adressen (zur technischen Bereitstellung der Website)
  • Zugriffsdatum und -uhrzeit
  • Aufgerufene Seiten
  • Technische Browserdaten

Diese Daten werden ausschließlich zur technischen Bereitstellung der Website verwendet und nicht mit anderen Datenquellen zusammengeführt. Die Rechtsgrundlage für die Datenverarbeitung ist Art. 6 Abs. 1 lit. f DSGVO.

Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt, einem Service der GitHub Inc. (USA). Bei jedem Zugriff auf diese Website werden daher automatisch Daten an Server in den USA übertragen. Wir weisen darauf hin, dass die USA kein sicherer Drittstaat im Sinne des EU-Datenschutzrechts sind.

4. Datenerfassung auf dieser Website

Server-Log-Dateien

Der Provider der Seiten erhebt und speichert automatisch Informationen in so genannten Server-Log-Dateien, die Ihr Browser automatisch übermittelt. Dies sind:

  • Browsertyp und Browserversion
  • verwendetes Betriebssystem
  • Referrer URL
  • Hostname des zugreifenden Rechners
  • Uhrzeit der Serveranfrage
  • IP-Adresse

Eine Zusammenführung dieser Daten mit anderen Datenquellen wird nicht vorgenommen.

5. Schriftarten

Diese Website verwendet selbst gehostete Schriftarten. Es werden keine externen Server kontaktiert, um Schriftarten zu laden. Die Schriftdateien werden direkt von diesem Webserver bereitgestellt.

Hinweis zum Hosting-Standort

Diese Website wird über die Server unseres Hosting-Anbieters in verschiedenen Ländern bereitgestellt. Wir haben sichergestellt, dass keine personenbezogenen Daten durch uns erfasst oder gespeichert werden.
