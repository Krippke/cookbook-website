So funktioniert das Teilen von Rezepten

Was ist eine .recipe Datei?

Eine kleine Datei, die alle Informationen eines Rezepts enthält - von den Zutaten bis zur Zubereitung. Perfekt zum Teilen mit Freunden und Familie.

Für Entwickler

Das .recipe Format ist offen und einfach zu implementieren

Einfache Struktur

// Tatsächliches Format
{ "version": 2, "text": "Käsespätzle\n\n500g Mehl\n2 Eier\n\nAlles mischen...", "image": null }

Vorteile

  • • Minimalistisches JSON-Format
  • • Intelligente Texterkennung beim Import
  • • Originaltext bleibt erhalten
  • • Optional: Bilder als Base64

Format Spezifikation

FeldTypBeschreibungPflicht
versionnumberFormat-Version (aktuell: 2)✓
textstringKompletter Rezepttext (Name, Zutaten, Schritte)✓
imagestring|nullBase64-kodiertes Bild (optional)
